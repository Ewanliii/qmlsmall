# Classical ML Baselines

This folder contains classical baseline models used for comparison with VQC variants.

## Script

- `ml_work.py`
  - Runs Logistic Regression, Random Forest, and MLP.
  - Uses the same fixed small-sample datasets as the QML experiments where possible.
  - Reports evaluation metrics for baseline comparison.

## Inputs

Recommended input:

- a fixed small dataset `.npz` containing `X` and `y`.

Alternative local input:

- full latent arrays `Z` and labels `y`, generated by preprocessing.
