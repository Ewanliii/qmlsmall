# Notebooks

This directory contains exploratory notebooks used during experiment development and analysis.

## Notebooks

- `DAE_trainer.ipynb`
  - Exploratory notebook for DAE training and preprocessing checks.

- `ml_work.ipynb`
  - Classical machine learning baseline experiments.

- `VQC_work.ipynb`
  - VQC experiments and circuit variant exploration.

- `drawplot.ipynb`
  - Plotting and visualization of experimental results.

## Note

The notebooks are provided for transparency. For cleaner execution and reproducibility, use the standalone scripts in [`src/`](../src/).
