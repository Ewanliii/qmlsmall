# Data

This directory documents the data used in the experiments.

## Dataset Source

The project uses the IEEE-CIS Fraud Detection dataset from Kaggle:

https://www.kaggle.com/competitions/ieee-fraud-detection/data

The original dataset is not included in this repository because it cannot be redistributed under the dataset license.

## Local Generated Artifacts

The preprocessing scripts generate local artifacts such as:

- `Z_latent_D8.npy`, `Z_latent_D16.npy`, `Z_latent_D32.npy`: latent feature matrices generated by the denoising autoencoder;
- `y_labels_*.npy`: labels aligned with latent features;
- `dae_encoder_*.pt`: trained DAE encoder weights;
- `imputer.joblib`: missing-value imputation artifact;
- `scaler.joblib`: standardization artifact;
- `Xy_small_*.npz`: fixed small-sample datasets used for controlled experiments.

Large generated files are ignored by Git and should not be committed.

## Reproducibility Note

The original Kaggle dataset should be downloaded manually. After placing it in a local path, follow the preprocessing scripts under [`src/preprocessing/`](../src/preprocessing/) to generate latent features and small experimental datasets.
